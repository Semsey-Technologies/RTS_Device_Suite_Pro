# ThemeEngineScreen.kt

## What it does
Provides a comprehensive interface for customizing the app's visual identity.

## Why it does it
To allow users to personalize their experience by selecting from dozens of "Cyber", "Minimal", and "Experimental" themes, as well as adjusting UI scaling and typography.

## How it does it
- **Preset Browser**: Displays categories of themes (e.g., "Cyber / Tech Core") using a horizontal scroll of preview cards.
- **Real-time Preview**: Applying a theme instantly updates the screen's background, accent colors, and text styles via the `ThemeManager`.
- **Advanced Controls**: Includes sliders for UI scaling (Title size, Body size) and switches for bold/italic text.
- **Font Selector**: A dropdown menu to switch between system font families (Monospace, Serif, etc.).

## Overall role
It is the personalization hub of the app, giving users full control over the aesthetic and accessibility of the interface.
