# ToolsScreen.kt

## What it does
A secondary hub for utility-focused features and advanced diagnostics.

## Why it does it
To organize miscellaneous power-user tools like the Smart Organizer, Log Exporter, and Resource Monitor into a single, easy-to-browse location.

## How it does it
- **Vertical List**: Presents a list of "Power Tools" with detailed descriptions and unique icons.
- **Categorization**: Groups tools by their function (e.g., "Automation", "Diagnostics", "File Management").
- **Direct Navigation**: Triggers navigation to specialized sub-screens when a tool is selected.

## Overall role
It acts as the "Advanced" menu for the application, hosting specialized utilities that extend the core backup and cleaning functionality.
