# icon.jpeg

## What it does
The source master image for the RTS Device Suite Pro brand logo.

## Why it does it
It serves as the high-fidelity source for the application's visual identity. It is referenced by `ic_launcher_foreground.xml` to generate the launcher icon and may be used elsewhere in the UI where a high-resolution logo is required.

## How it does it
- **Format**: JPEG (Joint Photographic Experts Group).
- **Usage**: Referenced as `@drawable/icon` in XML layouts and drawables.
- **Integration**: Placed within the `res/drawable` folder to be accessible via the standard Android resource system.

## Overall role
The central branding asset of the application. Its quality determines the look of the icon on the user's home screen.
